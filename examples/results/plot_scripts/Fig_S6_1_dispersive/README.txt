
To reproduce Figure S.6.1:
-------------------------

To acquire data:

Run the examples in the "ACE/examples/06_dispersive/" folder and copy the *.out files to this directory


To produce plot, run:

gnuplot figS6_1_data.plt && pdflatex figS6_1_data.tex && pdflatex figS6_1_resub.tex


Output:
figS6_1_resub.pdf
